# Flights Application
The project is a simple CRUD service for buying flight tickets.

## Server:
* Java 16
* Spring Boot
* Hibernate
* PostgreSQL Database
* Gradle

## Client: 
* Python 3.10
* PyQT5

## Testing:
* Mockito
* MockMVC
